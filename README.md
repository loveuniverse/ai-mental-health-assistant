# 心理健康AI助手后台管理系统

## 项目简介

心理健康AI助手是一个专注于心理健康领域的管理系统，出于时间和精力上的考量，目前仅实现了后台管理端，用于管理知识文章、咨询记录、情绪日志等数据，并提供数据分析功能。

## 技术栈

- **前端框架**：Vue 3
- **路由管理**：Vue Router 4
- **状态管理**：Pinia 3
- **UI组件库**：Element Plus
- **富文本编辑器**：WangEditor 5
- **图表库**：ECharts 6
- **网络请求**：Axios
- **构建工具**：Vite 8
- **样式预处理器**：Sass

## 核心功能

### 1. 数据分析

- 系统概览数据展示
- 关键指标统计分析

### 2. 知识文章管理

- 文章分类管理
- 文章列表展示与分页
- 文章创建、编辑、删除
- 文章状态管理
- 文章封面上传

### 3. 咨询记录管理

- 会话列表展示
- 会话详情查看
- 聊天记录管理

### 4. 情绪日志管理

- 情绪日志列表展示
- 情绪日志删除

### 5. 用户认证

- 管理员登录
- 管理员注册
- 权限控制

## API接口

系统通过以下主要接口与后端进行交互：

- **用户认证**：`/user/login`, `/user/logout`
- **知识文章**：`/knowledge/category/tree`, `/knowledge/article/page`, `/knowledge/article`
- **咨询记录**：`/psychological-chat/sessions`
- **情绪日志**：`/emotion-diary/admin/page`
- **数据分析**：`/data-analytics/overview`

## 配置说明

- **API基础路径**：`/api`
- **文件服务地址**：`http://159.75.169.224:1235`
- **请求超时时间**：5000ms

## 权限控制

系统通过路由守卫实现权限控制，只有管理员（userType == 2）可以访问后台管理页面。

## 注意事项

- 本项目目前仅实现后台管理端，未实现前台用户端
- 所有API请求需要携带token进行身份验证
- 文件上传功能已集成，支持文章封面上传
